Welcome

Thank you for participating to this project.
You are going to annotate with specific tags a set of 1000 tweet pairs in English.
The pairs consist in two versions of the same tweet. In the first version an emoji is missing; the second version is complete. According to the information added by the emoji you will select the most appropriate tag within the following list:

  • Redundant
  • Non Redundant
  • Non Redundant + POS
Please refer to the guidelines provided for a thorough explanation of the tags' meaning.
By clicking on the below button "GO TO EXAMPLES" you will be directed to the next page which will show you an example pair with the correspondent selected class.
To move to another example click on the "NEXT" button at the bottom of the page.
You will find a "START ANNOTATION" button on the bottom of the last example page .
